TI-tools

A compilation of calculator programs and tools, written by me in TI-BASIC.

Welcome to TI-tools.

This page is a quick place to download all the TI-BASIC tools I've written for the calculator over the years. It is not regularly maintained or updated but merge requests are by all means encouraged. Sometimes I do forget to upload new programs I've written but usually you'll find them sometime before or after a test in math or science classes.

Supported Calculators

As I've progressed through using friends' calculators and through several of my own calculators, the calculators these programs are for will tend to change. These programs are categorized by the calculator it was written for, but it is completely possible to use them on any compatible calculator. Currently the only calculator I support is the TI-84 Plus CE.

Adding programs yourself

It is quite simple to add a program yourself, simply drag and drop it into the corresponding TI Connect software for your calculator. If you would like to contribute to this repository, email me at the bottom of the page (I will give you credit) and I can add your program. In addition, you can clone the main branch and merge with it later.

Authors and Contributors

Gideon Tong (@gideontong) is currently the sole maintainer and contributer to the TI Tools repository. Programs may have been based on other programs but all source code is original. Since 2014, several programs have been written in order to help people get those A's on their tests.
